# Audit Check

## Purpose
- 工程監査、協力会社監査、4M 変更確認、帳票確認のチェック観点を統一する。

## Procedure
1. 対象工程と監査目的を確認する。
2. 工程フローから、実物、設備、記録を対応づける。
3. 識別、条件、測定、教育、異常時対応を順に確認する。
4. 合否判定と是正依頼事項を分けて整理する。
5. 追加で取得すべき証拠を残す。

## Checklist
- 工程フロー
- 識別管理
- 設備管理
- 条件管理
- 測定・検査
- 教育
- 異常時対応
- 記録
- 合否判定
- 是正依頼事項
- 追加取得すべき証拠

## Rules
- 不適合と観察事項を混ぜない。
- 写真、帳票、ログのどれで確認したかを残す。
- 4M 変更は、変更点、承認、反映記録の 3 点を最低確認する。
