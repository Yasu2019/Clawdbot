Notes

Direct STEP import is not included in this MVP. Use the existing STEP to 3D HTML or STEP to STL workflow first, then load the converted model here. DXF preview reuses the existing dxf2step API SVG preview.
Protocol: The supervised workflow reference is `local_llm_supervision_protocol_complete_utf8_bom.zip`. Key working rule: keep datum and GD&T data reviewable and editable rather than fully automatic.
