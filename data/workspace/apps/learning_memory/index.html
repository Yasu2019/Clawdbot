Quick Presets

この画面は利用可能な learning_engine の URL を自動検出します。 learning_engine が起動していれば、ポータルから直接開いてそのまま使えます。

Compare Email Thread

メールの件名、要約、未解決事項、次アクションから、似た thread を直接比較します。

Compare Case

不具合事例や品質案件を構造化して投入し、似た defect case を直接比較します。

Compare Quality Issue

品質問題のタイトル、症状、推定原因、暫定対応、是正内容をまとめて入力し、近い quality issue を探します。

Compare Improvement Activity

改善テーマ、実施内容、狙い、効果をまとめて入力し、近い improvement activity を探します。

Compare CAE Run

CAE 実行条件や失敗シグネチャを使って、似た simulation run を直接比較します。
