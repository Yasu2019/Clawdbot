# THREE.JS AUTONOMY PROTOCOL (TAP-2026)

> **Purpose:** Enable Clawdbot to autonomously verify and fix Three.js code generated by Antigravity.
> **Mechanism:** Puppeteer-based "Judge" script that captures logs and screenshots.

## 1. Roles

- **Coder (Antigravity):** Generates `index.html` with Three.js code.
- **Judge (Clawdbot):** Runs `verify_threejs.js` to inspect the page.
- **Feedback:** Judge returns `report.json` (Logs + Screenshot Path). Coder analyzes it and fixes bugs.

## 2. Directory Structure

```
projects/threejs_viz/
├── index.html          # Current Code (Target)
├── verify_threejs.js   # The Judge Script
├── report.json         # Verification Result
└── screenshots/        # Evidence (screen_TIMESTAMP.png)
```

## 3. The Judge Script (`verify_threejs.js`)

**Requirements:**

- Launch Headless Chrome (Puppeteer).
- Load `file://.../index.html`.
- **Check 1: Console Errors:** Capture any `console.error` or exceptions.
- **Check 2: WebGL Context:** Verify `gl.getError()` is clear.
- **Check 3: Visuals:** Take a screenshot after 1000ms.
- **Output:** Write `report.json` with status "PASS" or "FAIL".

## 4. Operational Loop

1. **Generate:** Antigravity writes `index.html`.
2. **Verify:** Clawdbot runs `node verify_threejs.js`.
3. **Result:**
    - **Pass:** Screenshot shows content, Logs empty. -> **DONE**.
    - **Fail:** Logs show error OR Screenshot is black. -> **RETRY**.
4. **Fix:** Antigravity reads `report.json` and updates `index.html`.

## 5. Environment

- **Node:** Container standard node.
- **Puppeteer:** Installed locally in `projects/threejs_viz`.
- **System Deps:** `xvfb`, `libgl1` (Already installed in Dockerfile).
